# Handy基础库

该库提供了基础的页面布局及功能解决方案

---

## 组成

* reset 页面重置

    包括样式重置、JS方法重置，提供部分Javascript1.6特性

* detect 系统、浏览器及特性检测

    ua检测和特性检测分拆开，按需依赖 

* layout 页面布局

    提供flexbox的自适应布局方式

* url 

    提供页面路由、history处理功能

* plugin 插件

    提供scroll transform transition相关功能插件，为layout服务，也可以单独依赖使用。
